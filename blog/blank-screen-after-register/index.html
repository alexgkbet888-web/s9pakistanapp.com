S9 blank screen after register

Blank screens after registration need device, cache, storage, and app version checks.

Quick checklist

  1. Wait for the dashboard to load before closing the app.
  2. Check storage, network, and app permissions.
  3. Clear cache only after recording the registration step.
  4. If blank screen repeats, document device and version source.
